About

私たちについて

車に関わるすべてのこと、それが私たちのミッション。

はじめまして、株式会社 D-MASH LINE です。

この度は、弊社に興味を持ってくださり、誠にありがとうございます。 私たちは、2021年より北海道白老町にて、中古車の販売・買取事業をスタートさせました。（Garage MASH）
おかげさまで、多くのお客様にご愛顧いただき、2024年11月から軽貨物運送事業もスタートする運びとなり、さらなる発展を目指し法人化する運びとなりました。

これからも、皆さまに満足いただけるサービスを提供できるように努めてまいります。 今後ともよろしくお願い申し上げます。

Works

事業内容

中古車の買取

お客様の大切なお車の『買取価格アップ』実現の為に、徹底的なコスト管理と運営を行っています。 自信を持ってお見積りさせていただきますので、お気軽にお問い合わせください。

中古車の販売

中間業者を通さないオークション直仕入れのため、高品質な中古車をお得な価格で販売できます。 お客様の条件にあった、お車をご提案させていただきますので、お気軽にお問い合わせください。

軽貨物運送

関東エリアを中心に多くの企業様から、ご依頼を頂き小口貨物の配送を行っています。 常温、冷蔵、冷凍、スポット便と幅広く対応しておりますので、お気軽にお問い合わせください。

中古車のリース

月々定額のお支払いで車に乗れるサービスです。 自家用車から商用車まで幅広く対応しておりますので、お気軽にご相談ください。

メンテナンス

ご購入後も安全なカーライフをお過ごし頂けるように、定期点検や整備、代車手配を承ります。 お客様のお車のコンディションを最適に保つために、ぜひご利用ください。

ペットレンタカー

愛するペットと一緒に旅行やお出かけを楽しみたいという方々のニーズにお応えするために、現在、事業開始に向けて準備中です。
